# Japan Study

일본어 학습을 위한 웹 애플리케이션입니다.

## 주요 기능

- 히라가나 학습
- 카타카나 학습
- 단어 학습
- 문장 학습
- 음성 발음 지원
- 학습 진도 추적

## 기술 스택

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## 시작하기

1. 저장소 클론
```bash
git clone https://github.com/[your-username]/japanstudy.git
cd japanstudy
```

2. 의존성 설치
```bash
npm install
```

3. 개발 서버 실행
```bash
npm run dev
```

4. 브라우저에서 `http://localhost:3000` 접속
